# Librerías Externas

## ¿Cuáles son las principales características de Bootstrap que lo hacen ideal para el desarrollo de sitios web responsivos?

Dentro de las características principales de Bootstrap se encuentran:

- **Diseño Responsivo:** utiliza un sistema de cuadrícula flexible que permite que los elementos se ajusten automáticamente a diferentes tamaños de pantalla, garantizando una experiencia de usuario coherente en dispositivos móviles, tabletas y desktop.

- **Componentes Reutilizables:** ofrece una amplia biblioteca de componentes predefinidos, como botones, formularios, menús de navegación y modales,permitiendo a los desarrolladores integrar fácilmente elementos sin necesidad de escribir código desde cero.

- **Mobile First:** adopta un enfoque "móvil primero", lo que significa que se prioriza el diseño para dispositivos móviles antes de adaptar el contenido a pantallas más grandes.

- **Interactividad:** Bootstrap incluye componentes interactivos como carruseles y barras de progreso, mejorando la comunicación con el usuario y la usabilidad del sitio web

## ¿Cómo facilita Bootstrap la personalización de estilos en comparación con el desarrollo CSS desde cero?

En comparación con CSS, Bootstrap facilita la personalización de estilos a través de varias características claves:

1. **Clases Utilitarias:** proporciona clases CSS utilitarias que permiten aplicar estilos rápidamente a los elementos HTML, lo que acelera el proceso de diseño y reduce la necesidad de escribir CSS adicional.

2. **Herramienta de Personalización:** Bootstrap incluye una herramienta de personalización que permite ajustar temas y estilos sin necesidad de modificar el código fuente directamente, facilitando la adaptación a las necesidades específicas del proyecto.

3. **Sistema de Variables:** utiliza variables Sass que permiten modificar fácilmente colores, fuentes y otros estilos globales. Esto simplifica la personalización y asegura consistencia en todo el proyecto.

## ¿Qué es jQuery y cuál es su principal objetivo en el desarrollo web?

**jQuery** es una biblioteca de JavaScript que simplifica la manipulación del DOM y la creación de páginas web interactivas y dinámicas. Su principal objetivo es facilitar el desarrollo web al permitir a los desarrolladores escribir menos código para realizar tareas complejas (aplicar animaciones, manejar eventos, entre otros), de manera más eficiente y con una sintaxis sencilla. Es importante resaltar, que jQuery asegura que el código funcione de manera consistente en diversas plataformas, lo que mejora la experiencia del usuario y acelera el proceso de desarrollo.

---
## ¿Por qué se dice que jQuery permite “escribir menos y hacer más”?

Se dice que Jquery permite "escribir menos y hacer más" porque simplifica significativamente las tareas comunes de programación en JavaScript, permitiendo a los desarrolladores lograr resultados complejos con menos líneas de código.
